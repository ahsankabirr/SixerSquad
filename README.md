# 🏏 SixerSquad

A fantasy cricket team builder where you earn coins, browse players, and assemble your ultimate squad of six.

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- React-Toastify
- JSON (local data)
- Vite

---

## ✨ Key Features

1. **Coin-Based Player Bidding** — Earn coins by clicking the banner button and spend them to buy players. If you can't afford a player, you'll be notified instantly.

2. **Smart Player Selection with Validation** — Prevents duplicate picks, enforces a 6-player squad limit, and gives real-time feedback via toast notifications.

3. **Live Squad Management** — Switch between Available and Selected views, track your squad count, and remove players to rebuild your lineup on the fly.

---

## 📌 Project Description

SixerSquad is a React-based fantasy cricket app. Users collect coins, browse a roster of international cricketers, and build a squad of up to six players. The app handles all edge cases — insufficient funds, duplicate selections, and squad limits — with clean toast alerts instead of browser popups.
